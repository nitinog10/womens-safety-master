# Women's Safety Master

An Android application designed to enhance women's safety through location tracking and emergency alerts.

## About

This app provides essential safety features for women, ensuring they have a reliable tool on their mobile devices during critical situations.

## Features

- Real-time location tracking
- Emergency alert notifications
- User-friendly interface with introductory screens
- Detailed user information management
- Background services for continuous location monitoring

## Tech Stack

- Java
- Android SDK
- Gradle

## Getting Started

### Prerequisites

- Android Studio
- Java Development Kit (JDK)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/nitinog10/womens-safety-master.git
   ```
2. Open the project in Android Studio.
3. Sync the project with Gradle files.

### Running the App

1. Connect an Android device or start an emulator.
2. Run the app from Android Studio.

## Project Structure

- `/app`: Main application code.
- `/gradle`: Gradle configuration files.
- `/res`: UI resources (layouts, drawables, strings).
